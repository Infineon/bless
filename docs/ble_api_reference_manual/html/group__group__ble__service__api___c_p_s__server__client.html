◆ Cy_BLE_CPS_RegisterAttrCallback()

void Cy_BLE_CPS_RegisterAttrCallback ( cy_ble_callback_t  callbackFunc)

Registers a callback function for service-specific attribute operations.

Service-specific Write Requests from the peer device will not be handled with an unregistered callback function.

Parameters
callbackFuncAn application layer event callback function to receive events from the PSoC 6 BLE Middleware. The definition of cy_ble_callback_t for CPS service is:
typedef void (* cy_ble_callback_t) (uint32_t eventCode, void *eventParam), where:
  • eventCode: Indicates the event that triggered this callback (e.g. CY_BLE_EVT_CPSS_NOTIFICATION_ENABLED).
  • eventParam: Contains the parameters corresponding to the current event; (e.g. pointer to cy_stc_ble_cps_char_value_t structure that contains details of the characteristic for which the notification enabled event was triggered).
